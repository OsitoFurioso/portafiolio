Semana 3

JavaScript

24 de Abril

Aprendimos que JavaScript es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Fue diseñado originalmente para ser utilizado en el desarrollo de aplicaciones web interactivas, brindando a los sitios web la capacidad de realizar acciones y responder a la interacción del usuario en tiempo real. JavaScript se ejecuta en el lado del cliente, lo que significa que el código se ejecuta en el navegador web del usuario final. Sin embargo, también se puede utilizar en el lado del servidor utilizando plataformas como Node.js. Esto permite a los desarrolladores crear aplicaciones web completas y escalables tanto en el lado del cliente como en el lado del servidor utilizando el mismo lenguaje.

Semana 5

REACT

8 de Mayo
  • React es una biblioteca de JavaScript de código abierto utilizada para construir interfaces de usuario (UI) interactivas y reutilizables. Desarrollada por Facebook, React se centra en la creación de componentes reutilizables que representan partes específicas de la interfaz de usuario.
  • La principal característica de React es su enfoque en el rendimiento y la eficiencia. Utiliza un modelo de programación basado en componentes, lo que significa que las interfaces de usuario se dividen en componentes individuales, cada uno con su propio estado y comportamiento. Estos componentes pueden ser anidados y reutilizados, lo que facilita el mantenimiento y la actualización de la interfaz.
  • En resumen, React es una biblioteca JavaScript popular y poderosa que simplifica el desarrollo de interfaces de usuario interactivas y reutilizables. Su enfoque en componentes, rendimiento y eficiencia ha hecho que sea ampliamente adoptado en la industria y es utilizado por numerosas empresas y desarrolladores para crear aplicaciones web modernas y escalables.

Instalación de React+Vite

  • Instalar Node de la siguiente página https://nodejs.org/en
  • Usar el siguiente linea de comando npx create-react-app mi-proyecto
  • Navegar al directorio del proyecto: Después de que se haya creado el proyecto, ve al directorio del proyecto ejecutando el siguiente comando: "cd mi-proyecto "
  • Iniciar el servidor de desarrollo: Una vez dentro del directorio del proyecto, ejecuta el siguiente comando para iniciar el servidor de desarrollo de React: "npm start "
  • Esto iniciará el servidor de desarrollo y abrirá tu aplicación de React en tu navegador por defecto en la dirección http://localhost:3000. Ahora podrás ver los cambios en tiempo real mientras trabajas en tu proyecto.
